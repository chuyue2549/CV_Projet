# DE-BUT (Dispositif d’Entraînement pour le BUT)

DE-BUT est un dispositif d’entraînement destiné aux étudiants du BUT.
Il permet la consultation de cours, la réalisation de QCM et la gestion des contenus via une interface administrateur.
